# 📽️ Movie Recommendation System (Content Based Filtering)
A content-based movie recommender using TF-IDF, cosine similarity, and an interactive Streamlit interface.

Live Demo → https://movie-recommendation-system-tmc5lhfeyzpausrk6vdns9.streamlit.app/

GitHub Repo → https://github.com/AdrirajDas00/movie-recommendation-system

Dataset Link → https://www.kaggle.com/datasets/rounakbanik/the-movies-dataset

<br>
🚀 Features:
<br><br>

🔍 Content-Based Filtering using 5 textual movie attributes (movie overview, genres, cast, director and production companies)

📐 TF-IDF Vectorization for semantic understanding

🧮 Cosine Similarity across 5000+ movies

⚡ Fast recommendations served from pre-computed similarity matrix

🎨 Modern Streamlit UI with custom theme (inspired by OTT UI)

📊 Movie insights (genres, ratings, overview, etc.) + IMDB link

<br><br>
🧪 Example Recommendation:

If you select “The Dark Knight”, the model uses cosine similarity to find movies with similar:

1) Plot structure

2) Themes/ genres

3) Cast / director/ production companies involved

4) Keywords
